# 邮局报刊订阅系统 — 部署说明（给其他电脑用）

> 本系统为「本地部署」的数据库课程作业，需在每台电脑上各装一份环境后运行。
> 按以下 4 步操作即可在自己的电脑上跑起来。

---

## 环境要求（必须先装好）

### 1. 安装 Python 3.10 或更高版本
- 下载：https://www.python.org/downloads/
- **安装时务必勾选「Add Python to PATH」**（很重要，否则启动脚本找不到 Python）
- 验证：打开 cmd，输入 `python --version`，能看到版本号即可

### 2. 安装 MySQL 8.0
- 下载：https://dev.mysql.com/downloads/installer/ （选 MySQL Community Server 或 Installer）
- 安装时记住你设置的 **root 密码**（后面要用）
- 安装完成后，MySQL 服务（MySQL80）应处于"正在运行"状态
- 验证：打开 cmd，输入 `mysql -u root -p`，能进去即可

### 3. 安装 pymysql（Python 连 MySQL 的库）
- 打开 cmd，执行：
  ```
  pip install pymysql
  ```
- （如果你用启动脚本 `start_all.bat`，这步会自动完成，可跳过）

## 启动方法

### 方法一：一键启动（推荐）

双击项目根目录的 **`start_all.bat`**，它会：
1. 自动检测 Python（若没装会报错提示）
2. 自动安装 pymysql（若已装则跳过）
3. **首次运行会提示输入 MySQL root 密码**（输入时不显示，正常现象），密码会存到 `.db_pwd` 文件，之后再启动不用重输
4. 启动后端服务（端口 8088）
5. 启动前端页面（端口 8000）
6. 自动打开浏览器

看到浏览器打开登录页后，用 **`admin / admin123`** 登录即可。

### 方法二：手动分步启动

打开两个 cmd 窗口：

**窗口 1（后端）：**
```
python run_server.py
```
首次会让你输 MySQL root 密码，输入后看到「启动 API 服务：http://127.0.0.1:8088」即成功。

**窗口 2（前端）：**
```
python serve_frontend.py
```
看到「静态服务已启动: http://127.0.0.1:8000」即成功。

然后浏览器打开：**http://127.0.0.1:8000/index.html**

## 停止服务

- 一键启动方式：关掉弹出的「后端」「前端」两个黑色窗口
- 手动方式：在对应 cmd 窗口按 `Ctrl + C`

## 常见问题

### Q1：双击 start_all.bat 闪退 / 提示 Python not found
→ 没装 Python，或安装时没勾「Add to PATH」。重装 Python 并勾选该选项，或把 Python 安装目录加到系统 PATH。

### Q2：提示「无法连接 MySQL 服务 (2003)」
→ MySQL 没装或没启动。
- 检查：`Win+R` → `services.msc` → 找 `MySQL80`，确认状态是"正在运行"。
- 没装就先装 MySQL 8.0。

### Q3：提示「MySQL 用户名或密码错误 (1045)」
→ 输入的 root 密码不对。
- 删掉项目根目录的 `.db_pwd` 文件，重新双击 `start_all.bat`，输入正确密码。
- 忘记 root 密码：用 `tools/reset_mysql2.bat` 重置（右键以管理员身份运行）。

### Q4：浏览器打开是空白页 / 登录页打不开
→ 前端服务没启动成功。确认窗口 2（前端）显示「静态服务已启动」，且端口 8000 没被占用。

### Q5：登录提示「无法连接后端」
→ 后端没启动成功。看后端窗口的报错，通常是 MySQL 连不上（见 Q2/Q3）。

### Q6：改了 MySQL 密码后启动不了
→ 删掉 `.db_pwd` 文件，重新启动会重新要你输密码。

## 目录结构说明

```
项目根/
├── start_all.bat          ← 双击启动入口
├── run_server.py          ← 后端启动器
├── serve_frontend.py      ← 前端启动器
├── schema.sql             ← 建库脚本（首次启动会自动建库建表，一般不用手动跑）
├── 部署说明.md            ← 本文件
│
├── server/                ← 后端代码（Python 包）
│   ├── api.py             ← HTTP 接口
│   ├── core/authority.py  ← 权限/员工/日志
│   ├── customer/          ← 客户数据管理
│   ├── newspaper/         ← 报刊数据管理
│   ├── subscription/      ← 订阅管理
│   ├── stock/             ← 报刊入库管理
│   └── delivery/          ← 报刊发放系统
│
├── web/index.html         ← 前端页面
├── tools/                 ← 工具脚本（MySQL 密码重置等）
└── 流程图/                ← 设计图
```
